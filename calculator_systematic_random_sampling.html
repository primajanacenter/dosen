Kalkulator Besar Sampel
Systematic Random Sampling

Rumus Besar Sampel (Proporsi):
n = (Z² × p × (1 − p)) / d²

Rumus Interval Sistematik:
k = N / n

Keterangan:
Z = Nilai Z
p = Estimasi proporsi
d = Margin of error
N = Total populasi
k = Interval pengambilan sampel
